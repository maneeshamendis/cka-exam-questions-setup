## 🎯 About This Repository

This repository contains practice questions for CKA exam. Each folder has question and the answer separately. Also execute the yaml file on your cluster or killercoda playground (any playground) and practice the question.

---

## 📁 Repository Structure
📦 CKA-Practice-Questions
┣ 📂 [Topic-Name]/
┃ ┣ 📜 QUESTION.md
┃ ┣ 📜 ANSWER.yaml
┃ ┗ 📜 SOLUTION.md
┗ ...

text

---

## 🚀 How to Use

1. Navigate to any topic folder
2. Read QUESTION.md
3. Try to solve it yourself
4. Apply the solution: `kubectl apply -f ANSWER.yaml`
5. Check SOLUTION.md for explanations
