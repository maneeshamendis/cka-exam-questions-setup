## 📋 Overview

This repository is your **ultimate resource** for CKA exam preparation. It contains carefully crafted practice questions that mirror real exam scenarios, complete with automated environment setup.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🎯 **Real Exam Scenarios** | Questions designed to match actual CKA exam difficulty |
| 🤖 **Auto-Setup** | YAML files that instantly create the practice environment |
| 📝 **Separated Q&A** | Question and answer in different files for effective practice |
| 🚀 **Multi-Platform** | Works on any Kubernetes cluster or playground |
